Using Modules in Node.js - A Beginner's Guide

Add a Local Module to a Node.js Application

In Node.js, you can create your own local modules and add them to your application. A module is simply a JavaScript file that contains related code. To add a local module, you create a new file, write the necessary code inside it, and then import it into your main application.

Example: Let’s say you want to add a local module that performs basic math operations like addition and subtraction. First, create a file called math.js in your project’s directory: 


// math.js (local module)
function add(a, b) {
    return a + b;
}

function subtract(a, b) {
    return a - b;
}

Use the module.exports Property to Export from a Module

To make the functions or variables in a module accessible to other files in your application, you need to export them using the module.exports property. This allows other files to import and use the module’s content.

Example: In the math.js file, you can export the add and subtract functions so they can be used elsewhere in the application: 


// Export functions from math.js
module.exports = {
    add,
    subtract
};

Now, both functions are exported as part of an object, making them available for use in other parts of your project.

Use the require() Function to Import from a Module

To import functions or variables from a module into another file, you use the require() function. This function reads the exported content from a local module and makes it available for use in the current file.

Example: To use the add and subtract functions from the math.js module in your main application, import them using require(): 


// app.js (main application)
const math = require('./math');

const sum = math.add(5, 3);
const difference = math.subtract(5, 3);

console.log(`Sum: ${sum}`);          // Output: Sum: 8
console.log(`Difference: ${difference}`);  // Output: Difference: 2

Here, the require() function imports the add and subtract functions from the math module, and you can now use them in app.js.
