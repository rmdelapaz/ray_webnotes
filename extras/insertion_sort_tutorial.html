Understanding Insertion Sort in JavaScript

In this tutorial, we’ll explore another fundamental sorting algorithm: Insertion Sort. Insertion Sort is efficient for small datasets and works similarly to how you might organize cards in your hand. Let’s break it down and explain each step to deepen your understanding of this useful sorting method.

What is Insertion Sort?

Insertion Sort is a simple algorithm that builds the sorted array one item at a time. It’s like organizing playing cards: you pick up one card at a time, compare it to the cards in your hand, and place it in the correct position. The array gets sorted as you go through each element.

When and Why to Use Insertion Sort?

Insertion Sort is useful for small arrays or nearly sorted arrays because it’s relatively efficient in these cases. Its time complexity is O(n^2), so for larger arrays, algorithms like Merge Sort or Quick Sort are faster. However, due to its simplicity, Insertion Sort is often used when you want to implement a basic sorting technique.

Insertion Sort Code Example

        
let arr = [52, 12, 27, 13, 30];

function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let key = arr[i];
        let j = i - 1;

        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j = j - 1;
        }

        arr[j + 1] = key;
    }
    return arr;
}

console.log("Before sorting: ", arr);
console.log("After sorting: ", insertionSort(arr));

Code Explanation

Let’s go through the code step by step:

1. Initial Array Setup

The array we want to sort is defined as:

let arr = [52, 12, 27, 13, 30];

Our goal is to sort this array in ascending order.

2. The Outer Loop

The outer loop starts at the second element (index 1) and moves forward through the array:

for (let i = 1; i < arr.length; i++) {

Why start at index 1? Because we assume that the first element (index 0) is already sorted by itself, and we’ll insert the rest of the elements one by one.

3. The Key Element

For each iteration of the outer loop, we set the current element as the "key" that we want to insert into its correct position within the sorted part of the array:

let key = arr[i];

4. The Inner Loop and Shifting Elements

The inner loop moves elements that are larger than the key one position to the right. Here's how it works:


    while (j >= 0 && arr[j] > key) {
        arr[j + 1] = arr[j];
        j = j - 1;
    }

This loop keeps shifting elements to the right until we find the correct position for the key. For example, if we’re sorting arr = [52, 12, 27, 13, 30] and key = 12, the inner loop shifts 52 to the right, making room for 12 to be inserted in the correct spot.

5. Inserting the Key

Once we’ve shifted the larger elements, we place the key in the correct position:

arr[j + 1] = key;

This ensures the key is inserted where it belongs, leaving the previous elements in order and placing the key in the right spot.

6. Returning the Sorted Array

Finally, once the array is fully sorted, the function returns the sorted array:

return arr;

For our example, arr starts as [52, 12, 27, 13, 30] and is sorted into [12, 13, 27, 30, 52].

Real-World Example and Applications

Imagine you’re organizing books on a shelf. You pick up each new book and find its correct spot among the already organized books. This is similar to how Insertion Sort works: you go through the list, one item at a time, and place each item in its correct position within the sorted section.
