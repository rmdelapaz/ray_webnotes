Step 3: Carrying Out the Plan (Pseudocode & Solution)

Pseudocode:


// 1. Create a bubbleSort function
// 2. Iterate through the array comparing adjacent values
// 3. Swap values if the first is greater than the second
// 4. Repeat the process until the array is sorted

Basic Solution (Bubble Sort):

function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // Swap the elements
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    return arr;
}

// Expected input: [5, 3, 8, 4, 2]
// Expected output: [2, 3, 4, 5, 8]
console.log(bubbleSort([5, 3, 8, 4, 2]));

Explanation:

The bubbleSort function iterates through the array multiple times. On each iteration, it compares adjacent elements and swaps them if they are in the wrong order. This continues until the entire array is sorted.

Think of this like sorting books on a shelf by height. You start by comparing two books next to each other, and if the one on the left is taller, you swap them. You keep doing this until all the books are in the right order.

Advanced Solution (Optimized Bubble Sort):

function optimizedBubbleSort(arr) {
    let n = arr.length;
    let swapped;
    for (let i = 0; i < n - 1; i++) {
        swapped = false;
        for (let j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // Swap the elements
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
                swapped = true;
            }
        }
        // If no elements were swapped, the array is already sorted
        if (!swapped) break;
    }
    return arr;
}

// Expected input: [5, 3, 8, 4, 2]
// Expected output: [2, 3, 4, 5, 8]
console.log(optimizedBubbleSort([5, 3, 8, 4, 2]));

This optimized version stops iterating if no swaps were made during a pass, meaning the array is already sorted.
