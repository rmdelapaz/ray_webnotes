Understanding Selection Sort in JavaScript

In this tutorial, we'll dive into another fundamental sorting algorithm: Selection Sort. Like Bubble Sort, it's often used to teach the basics of sorting algorithms due to its simplicity. We will break down the algorithm step by step to ensure clarity for new developers.

What is Selection Sort?

Selection Sort works by repeatedly finding the smallest element from the unsorted portion of the array and placing it in its correct position. Think of it like sorting a stack of papers: you look through the pile, find the smallest piece, and put it at the top. You repeat this process until the entire pile is sorted.

When and Why to Use Selection Sort?

Selection Sort is often used for educational purposes because it is easy to understand and implement. However, like Bubble Sort, its time complexity is O(n^2), making it inefficient for large datasets. In real-world applications, we tend to use faster algorithms such as Merge Sort or Quick Sort, but for small datasets or where simplicity is key, Selection Sort works just fine.

Selection Sort Code Example

        
let arr = [5, 2, 7, 1, 0];

function selectionSort(arr) {
    for (let i = 0; i < arr.length - 1; i++) {
        let minIndex = i;
        for (let j = i + 1; j < arr.length; j++) {
            if (arr[j] < arr[minIndex]) minIndex = j;
        }
        let temp = arr[minIndex];
        arr[minIndex] = arr[i];
        arr[i] = temp;
    }
    return arr;
}

console.log("Before sorting: ", arr);
console.log("After sorting: ", selectionSort(arr));

Code Explanation

Let's walk through the code line by line:

1. Initial Array Setup

The array we want to sort is defined as follows:

let arr = [5, 2, 7, 1, 0];

Our goal is to sort this array in ascending order, from smallest to largest.

2. The Outer Loop

The outer loop determines the position of the sorted portion of the array: 

for (let i = 0; i < arr.length - 1; i++) {

Each time through this loop, we move the smallest unsorted element to its correct position in the sorted portion of the array. The loop runs arr.length - 1 times because we don't need to compare the last element.

3. Finding the Minimum Element

Within the outer loop, we use an inner loop to find the smallest element in the unsorted portion of the array: 

let minIndex = i;

This line initializes minIndex to the current index i, assuming the first unsorted element is the smallest. 

for (let j = i + 1; j < arr.length; j++) {
    if (arr[j] < arr[minIndex]) minIndex = j;
    }

Here, we compare each remaining unsorted element (arr[j]) to the current minimum (arr[minIndex]). If a smaller element is found, we update minIndex.

4. Swapping Elements

After finding the smallest element, we swap it with the first unsorted element (arr[i]): 


    let temp = arr[minIndex];
    arr[minIndex] = arr[i];
    arr[i] = temp;

This ensures that the smallest element is placed in its correct sorted position.

5. Returning the Sorted Array

After sorting the entire array, the function returns the sorted array: 

return arr;

For our example, arr starts as [5, 2, 7, 1, 0] and is sorted into [0, 1, 2, 5, 7].

Real-World Example and Applications

Imagine sorting books on a shelf by height. You start by looking for the smallest book, take it out, and place it in the first position. Then, you look for the next smallest book, and so on. This is similar to how Selection Sort works.
