Understanding Bubble Sort in JavaScript

In this tutorial, we'll be exploring the Bubble Sort algorithm, which is a simple sorting technique often used as an introduction to sorting algorithms. We'll go step by step and understand how the code works by breaking down the logic line by line. Think of this like a series of bubble movements—hence the name "Bubble Sort".

What is Bubble Sort?

Bubble Sort is a sorting algorithm that works by repeatedly stepping through the list to be sorted, comparing adjacent items, and swapping them if they are in the wrong order. This process is repeated until the entire list is sorted.

Imagine you're blowing bubbles in a glass of water. The largest bubbles will naturally float to the top. In Bubble Sort, the largest value in an array 'floats' to the end of the array with each pass, just like the largest bubbles in water.

When and Why to Use Bubble Sort?

Bubble Sort is often used as a teaching tool because of its simplicity. However, in real-world applications, it's not the most efficient for large datasets since its time complexity is O(n^2), meaning it can be slow for long arrays. Despite that, it's helpful for small datasets or situations where simplicity and clarity are prioritized over efficiency.

Bubble Sort Code Example

        
let arr = [13, 5, 9, 2, 27];

function bubbleSort(arr) {
    for (let i = 0; i < arr.length - 1; i++) {
        let swapped = false;
        for (let j = 0; j < arr.length - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }
        if (!swapped) break;
    }
    return arr;
}

console.log("Before sorting: ", arr);
console.log("After sorting: ", bubbleSort(arr));

Code Explanation

Let's break down the code step by step:

1. Initial Array Setup

We start with an array of numbers that we want to sort:

let arr = [13, 5, 9, 2, 27];

We want to sort this array in ascending order, from smallest to largest.

2. The Outer Loop

The outer loop in the bubbleSort function controls how many passes we make over the array. Here's the line of code that defines it:

for (let i = 0; i < arr.length - 1; i++) {

We loop through the array, reducing the number of elements to check each time since the largest element 'bubbles up' to its correct position after every full pass through the inner loop.

3. The Inner Loop and Swapping

The inner loop compares adjacent elements and swaps them if they're in the wrong order. Here’s the key part of the logic:


for (let j = 0; j < arr.length - i - 1; j++) {
    if (arr[j] > arr[j + 1]) {
        let temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
        swapped = true;
    }
}

Here’s how it works:

  • Comparison: The algorithm checks if the current element (arr[j]) is greater than the next element (arr[j + 1]).
  • Swapping: If the current element is greater, it swaps them. It uses a temporary variable (temp) to hold one value during the swap.
  • Flagging Swaps: If a swap happens, the swapped flag is set to true.

This process is repeated for every element in the array until the largest number 'bubbles up' to its correct position at the end of the array.

4. Optimization with the Swapped Flag

If no elements were swapped in a pass, the array is already sorted, and we can stop early. This is done with the following code:

if (!swapped) break;

This is an optimization that makes Bubble Sort more efficient if the array becomes sorted before all passes are complete.

5. Returning the Sorted Array

Once the array is sorted, the function returns the sorted array:

return arr;

In our example, arr starts as [13, 5, 9, 2, 27] and is sorted into [2, 5, 9, 13, 27].

Real-World Example and Applications

Imagine you're sorting a deck of cards in your hand. You repeatedly compare each card to the next one, swapping them if needed, just like the Bubble Sort algorithm does with numbers in an array. However, in real-world programming, we typically use faster algorithms like Merge Sort or Quick Sort for larger datasets, but Bubble Sort is great for educational purposes and small-scale problems.
